# My Cypress Automation

In this project, I began my studies with Cypress, a modern end-to-end testing tool. During this process, I learned how to set up the environment, create automated tests, and validate the application's behavior in a practical way.

## Installation 
```bash
npm install
```
> You must have Node.js installed


## Runnig
```bash
Open the Cypress UI
npx cypress open

or

run the Cypress 
npx cypress run
```
